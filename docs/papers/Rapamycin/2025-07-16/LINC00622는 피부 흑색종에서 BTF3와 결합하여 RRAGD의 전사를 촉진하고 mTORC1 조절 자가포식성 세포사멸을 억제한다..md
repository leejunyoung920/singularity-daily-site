# LINC00622는 피부 흑색종에서 BTF3와 결합하여 RRAGD의 전사를 촉진하고 mTORC1 조절 자가포식성 세포사멸을 억제한다.

**원제목:** LINC00622 transcriptionally promotes RRAGD to repress mTORC1-modulated autophagic cell death by associating with BTF3 in cutaneous melanoma

**요약:** LINC00622는 BTF3와 결합하여 피부 흑색종에서 mTORC1-조절 자가 세포 사멸을 억제하기 위해 전사적으로 RRAGD를 촉진합니다.
