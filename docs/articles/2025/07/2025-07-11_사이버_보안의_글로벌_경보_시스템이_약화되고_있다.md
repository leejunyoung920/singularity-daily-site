# 사이버 보안의 글로벌 경보 시스템이 약화되고 있다

**영문 원제목:** Cybersecurity’s global alarm system is breaking down  
**출처:** https://www.technologyreview.com/2025/07/11/1119370/cybersecurity-alarm-system-breaking-down/

## 요약
- 미국의 취약점 추적 시스템인 NVD(National Vulnerability Database)가 2024년 2월부터 새로운 항목 게시를 중단했고, CVE(Common Vulnerabilities and Exposures) 프로그램도 계약 만료 위기에 처했음
- 이로 인해 사이버 보안 전문가들 사이에서 긴급한 우려가 제기되었으며, CISA(Cybersecurity and Infrastructure Security Agency)가 임시 조치를 취했으나 근본적인 해결책은 마련되지 않음
- NVD의 예산 삭감과 CISA의 자금 지원 중단으로 인해 취약점 분석 지연 문제가 발생했고, CISA는 대안으로 Vulnrichment 프로그램을 시작했지만 장기적인 해결책은 아님
- 전반적으로 글로벌 사이버 보안 경보 시스템에 심각한 공백이 발생하고 있어, 사용자들의 디지털 생활이 더욱 취약해질 것으로 우려됨
