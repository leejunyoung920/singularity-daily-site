# 시간이란 무엇일까요?

**영문 원제목:** Does Anybody Really Know What Time Is?  
**출처:** https://nautil.us/does-anybody-really-know-what-time-is-1223272/

## 요약
- 시간의 개념은 오랫동안 인간을 궁금하게 해왔지만 정확하게 무엇인지 정의하기 어려웠음
- 신경과학자 Dean Buonomano는 시간 지각이 뇌의 신경 활동을 통해 이루어지며, 뇌가 시간의 흐름과 과거/미래 경험을 만들어낸다고 설명함
- 뇌에는 서로 다른 타임스케일의 시간 감각 메커니즘이 존재하며, 이를 통해 우리가 시간을 경험하고 활용할 수 있게 된다고 말함
- 이처럼 시간은 단순한 물리적 개념이 아니라 뇌가 만들어내는 주관적 경험이라고 볼 수 있음
- 시간 지각 능력은 인류 문명 발전에 핵심적인 역할을 해왔다고 저자는 설명함
