# Robotic Arm “Feels” Using Sound

**출처:** https://spectrum.ieee.org/farm-robots-sound-based-sensing

## 요약
5줄 요약:

1. 새로운 접근법인 SonicBoom을 통해 자율 로봇이 농업 환경에서 소리를 이용해 물체를 감지할 수 있습니다.
2. SonicBoom은 접촉 마이크로폰 배열을 사용하여 물체와의 접촉을 감지하고 cm 단위로 정확하게 위치를 파악할 수 있습니다.
3. 기존의 카메라 기반 촉각 센서와 압력 센서는 농업 환경에서 효과적이지 않지만, SonicBoom은 더 효과적이고 저렴한 솔루션을 제공합니다.
4. 접촉 마이크로폰은 로봇 팔 내부에 더 깊이 내장될 수 있어 외부 손상으로부터 보호될 수 있습니다.
5. SonicBoom은 작은 수의 마이크로폰만으로도 효과적으로 물체를 감지할 수 있어 더 저렴한 솔루션을 제공합니다.
